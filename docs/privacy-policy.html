Kebijakan Privasi

Terakhir diperbarui: 13 Maret 2026

Ringkasan: Catatan Keuangan adalah aplikasi offline. Kami tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda ke server manapun. Semua data tersimpan hanya di perangkat Anda.

1. Pendahuluan

Kebijakan Privasi ini menjelaskan bagaimana aplikasi Catatan Keuangan ("Aplikasi") yang dikembangkan oleh Ali Imran ("Pengembang", "Kami") menangani informasi pengguna. Dengan menggunakan Aplikasi ini, Anda menyetujui praktik yang dijelaskan dalam kebijakan ini.

2. Data yang Disimpan di Perangkat

Aplikasi menyimpan data berikut secara lokal di perangkat Anda menggunakan database SQLite:

  • Data Transaksi: Nominal, kategori, tanggal, catatan, dan foto bukti transaksi yang Anda masukkan secara manual.
  • Data Kategori: Kategori kustom yang Anda buat untuk mengelompokkan transaksi.
  • Foto Bukti: Gambar yang Anda lampirkan dari kamera atau galeri, disimpan di penyimpanan internal aplikasi.

⚠️ Semua data ini hanya tersimpan di perangkat Anda dan tidak pernah dikirim ke server atau pihak ketiga manapun.

3. Data yang TIDAK Kami Kumpulkan

Kami tidak mengumpulkan:

  • Informasi identitas pribadi (nama, email, nomor telepon)
  • Data lokasi
  • Informasi perangkat atau sistem operasi
  • Data penggunaan atau analitik
  • Cookie atau teknologi pelacakan
  • Data iklan atau advertising ID

4. Izin Aplikasi (Permissions)

Aplikasi meminta izin berikut, yang semuanya digunakan secara lokal:

  • Kamera (CAMERA): Untuk mengambil foto bukti transaksi langsung dari kamera. Foto disimpan hanya di penyimpanan internal aplikasi.
  • Penyimpanan (READ_MEDIA_IMAGES / READ_EXTERNAL_STORAGE): Untuk memilih foto dari galeri sebagai bukti transaksi.
  • Notifikasi (POST_NOTIFICATIONS): Untuk menampilkan pengingat harian mencatat pengeluaran. Fitur ini bersifat opsional dan dapat diaktifkan/dinonaktifkan di Pengaturan.
  • Boot Completed (RECEIVE_BOOT_COMPLETED): Untuk menjadwalkan ulang pengingat setelah perangkat di-restart.

Semua izin ini digunakan secara lokal dan tidak mengirim data ke server manapun.

5. Berbagi Data

Aplikasi memiliki fitur Ekspor CSV yang memungkinkan Anda mengekspor data transaksi ke file CSV. Proses berbagi ini sepenuhnya dikontrol oleh Anda — Anda memilih sendiri ke mana file tersebut dikirim (WhatsApp, Email, Google Drive, dll).

Kami tidak secara otomatis membagikan data Anda kepada pihak manapun.

6. Keamanan Data

Karena semua data disimpan secara lokal di perangkat Anda:

  • Data dilindungi oleh mekanisme keamanan bawaan sistem operasi Android.
  • Data hanya dapat diakses oleh aplikasi Catatan Keuangan.
  • Jika Anda menghapus aplikasi, semua data akan ikut terhapus secara permanen.

7. Penghapusan Data

Anda dapat menghapus data kapan saja melalui:

  • Hapus per transaksi: Buka detail transaksi → tekan tombol "Hapus"
  • Hapus semua data: Pengaturan → Hapus Semua Data
  • Uninstall aplikasi: Semua data otomatis terhapus

8. Layanan Pihak Ketiga

Aplikasi ini tidak menggunakan layanan pihak ketiga seperti:

  • Google Analytics atau Firebase Analytics
  • Facebook SDK
  • Jaringan iklan (AdMob, dll)
  • Crash reporting (Crashlytics, Sentry, dll)

9. Privasi Anak-Anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi dari anak-anak. Karena aplikasi ini tidak mengumpulkan data apapun, tidak ada risiko pengumpulan data anak.

10. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan dipublikasikan di halaman ini dengan tanggal pembaruan yang baru. Kami menyarankan Anda untuk meninjau halaman ini secara berkala.
